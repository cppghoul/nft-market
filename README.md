# 🎁 Telegram NFT Marketplace

NFT маркетплейс подарков для Telegram с мини-приложением.

## 🌟 Особенности

- 🔐 Авторизация через Telegram
- 🛍️ Покупка NFT подарков
- 💝 Дарение цифровых подарков
- 📱 Полная интеграция с Telegram Mini Apps
- 🚀 Хостинг на Railway

## 🚀 Быстрый старт

1. Клонируйте репозиторий
2. Установите зависимости: `npm install`
3. Настройте .env файл
4. Запустите: `npm run dev`
